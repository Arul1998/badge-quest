## GitHub Achievements Guide

GitHub awards achievement badges to users who complete certain actions on the platform.

| Badge | How to Earn | Difficulty |
|-------|------------|------------|
| **Quickdraw** | Close an issue or PR within 5 minutes of opening it | Very Easy |
| **Pull Shark** | Get 2 pull requests merged | Medium |
| **YOLO** | Merge a PR without a code review | Medium |
| **Galaxy Brain** | Get 2 answers accepted in GitHub Discussions | Medium |
| **Starstruck** | Have a repository receive 16+ stars | Hard |
| **Public Sponsor** | Sponsor an open-source project | Easy |
| **Pair Extraordinaire** | Co-author commits with another contributor | Medium |
